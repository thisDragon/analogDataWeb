3.巡检模拟需要在作业资源中配置对应终端和巡检时间,并且配置中巡检反馈内容配置至少为1个;
4.围栏告警模拟需要在作业资源中配置对应终端和告警类型;(在改动后可以不配置)
5.考勤模拟也需要在作业资源中配置对应的考勤,考勤数是根据配置数来决定的,考勤和未考勤的数目比1:2或1:3或1:4(在改动后可以不配置)
